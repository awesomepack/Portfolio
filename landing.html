Heirholzer's algorithm is used to find Eulerian circuits or paths in linear O(n) time. It has an interesting application in biology where it is used to determine the DNA sequence of a large fragment by finding the euler circuit between its smaller fragments. In this project we review the algorithm's application in DNA sequencing and use it to implement a solution that takes a list of DNA fragments and returns a DNA sequence.
